==================
google-music-proto
==================


google-music-proto is a `sans-I/O <https://sans-io.readthedocs.io/>`_
library for interacting with Google Music.


Getting Started
===============

Install google-music-proto with `pip <https://pip.pypa.io/en/stable/>`_.

.. code-block:: console

	$ pip install google-music-proto
